\# 🥇 Gold Price Predictor — ML Web App



A full-stack ML web app that predicts Gold (GLD ETF) prices using Random Forest.



\## 🌐 Live Demo

https://gold-price-predictor-livid.vercel.app



\## 🧠 Model

\- Algorithm: Random Forest Regressor

\- Tuning: GridSearchCV

\- Features: SPX, USO, SLV, EUR/USD, Year, Month, Day, Quarter

\- Target: GLD ETF Price



\## 🗂️ Project Structure

\- backend/ → Flask REST API + trained model

\- frontend/ → HTML/CSS/JS UI

\- GoldPricePrediction.ipynb → Full ML notebook



\## 🚀 Tech Stack

Python · Flask · Scikit-learn · Vercel · HTML/CSS/JS
